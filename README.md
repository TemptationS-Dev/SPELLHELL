# SPELLHELL
2D, Pixel Art, Medieval, Bosses Only Bullet Hell.
A medieval-occult boss-only bullet hell featuring a lone mage in a gauntlet of escalating arcane horrors.

#Features
Tight, readable bullet patterns
Hand-crafted boss encounters
Customizable spells
High-replay challenge structure

#Tech Stack
Engine: Godot
Art: Aseprite
Resolution: Pixel art, 32×32 base assets, bosses 64×64+

#Purpose
This repository documents the development of SPELLHELL from zero experience to a complete project. It includes progress logs, prototypes, experiments, and final builds.

#How to Build
Clone repo
Open with Godot
Run main scene
